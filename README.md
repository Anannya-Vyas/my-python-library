# StatTools

StatTools is a lightweight statistics library built from scratch in Python.

## Features

- Mean
- Median
- Percentile
- Interquartile Range (IQR)
- Outlier Detection

## Installation (Local)

Run inside the project folder:

pip install .

## Example

```python
from stattools import mean

data = [10, 20, 30]
print(mean(data))
